🔑 ใส่ Key จาก EmailJS

วิธีตั้งค่า EmailJS (ฟรี)
  1. ไปที่ emailjs.com แล้วสมัครบัญชีฟรี
  2. เพิ่ม Email Service (เลือก Gmail แล้วเชื่อม Google account)
  3. สร้าง Email Template โดยใส่เนื้อหาด้านล่างนี้
  4. คัดลอก Public Key, Service ID, Template ID มาใส่ในฟอร์มด้านบน
  5. กด บันทึก Config แล้วทดสอบส่ง Email
Template Variables ที่ต้องใส่ใน EmailJS Template:

Subject: {{subject}}
To Email: {{to_email}}
Body (HTML): {{html_message}}

⚠ สำคัญ: ในช่อง Body ของ EmailJS Template ให้เลือก HTML mode แล้วใส่เพียง {{html_message}} เพื่อให้ email แสดงผลเป็น HTML ได้ครับ
